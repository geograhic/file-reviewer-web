# 智能文件复习系统 2.0 WebUI

这是独立新版本，不会修改 1.x 原项目文件夹。当前应用版本为 `2.14.0`，默认以桌面窗口运行，不再像普通网页一样弹浏览器，也不会显示黑色命令行窗口。

## 启动

- 打包版：双击 `dist\智能文件复习系统2.14.0_WebUI.exe`
- 源码版：双击 `启动_智能文件复习系统2.0.bat`
- 调试浏览器模式：`python app.py --browser`
- 纯本地服务模式：`python app.py --no-window --port 8897`

应用内部只监听本机 `127.0.0.1`。如果默认端口 `8765` 被占用，会自动尝试后续端口。

## 数据与配置

默认主数据目录：

`C:\Users\你的用户名\Documents\LiFileReviewer2`

所有关键数据集中放在这个目录里：

- `config.json`：用户配置
- `review_data.sqlite`：复习数据库
- `backups\`：数据库备份
- `plugins\`：插件目录，支持外部 JSON 清单；内置插件可在设置页直接启用或关闭
- `notes\`：默认笔记目录，保存真实 Markdown 文件
- `exports\`：默认导出目录。所有导出/备份都会先让用户选择保存位置；这个目录用于保存窗口的默认起点
- `app.log`：运行日志
- `runtime.json`：本次启动的本地服务信息

设置页支持直接自定义“主数据目录”“笔记目录”和“默认导出目录”，也支持一键导出迁移包 `LiFileReviewer2_profile_*.zip`。主数据目录可以迁移到空文件夹；如果选择了已有普通文件夹，软件会在里面创建 `LiFileReviewer2` 专用子目录，避免混入用户原有文件。迁移后，默认目录里只保留一个很小的 `profile_location.json`，用于下次启动时找到新的主数据目录。

Windows 深层目录和长文件名会自动启用长路径兼容。关联资料创建笔记时，软件会尽量保留完整标题；只有当文件系统拒绝过长文件名时，才会自动缩短落盘文件名，笔记标题和内容仍保留完整信息。

## 主要能力

- 直接选择本地文件夹作为文件库，像 Obsidian 一样浏览已有资料
- 文件库列表可删除失效库或不再需要的库；删除只清理软件索引，不删除原始文件
- 也可以直接添加单独文件，不必先选择或扫描文件夹
- 如果原生文件夹选择器不可用，可直接粘贴文件夹路径并扫描
- 批量扫描 PDF、Word、PPT、Excel、Markdown、HTML、图片、音视频等资料
- 像 Anki 一样自定义复习卡组/分类，支持默认卡组重命名、次级卡组、拖拽调整层级、按卡组筛选和批量分配
- 批量设置标签、暂停、恢复、完成、设为今天复习、删除索引
- 成就系统作为“成就系统”插件运行，可在插件管理中启用或关闭，并提供 XP、等级、阶段称号
- 成就可通过 `plugins\...\plugin.json` 扩展为无限里程碑，默认只读 JSON，不执行第三方代码
- 设置页插件管理支持导入插件 zip 包、导入插件文件夹、一键打开插件目录
- 新增“学习统计”插件，可视化展示复习次数、学习时长、卡组进度、内容关联和近期学习趋势；关闭插件后总览页模块会移除
- 新增“社交资料”插件，可保存昵称、账号 ID、简介、主页、分享偏好，并生成社交名片 JSON，为后续好友、动态、协作学习扩展预留接口
- 可导出分享包，默认包含索引、卡组、标签、关联笔记，也可选择包含原始文件；导出前可自定义保存路径
- 可在资料预览或笔记编辑中选中文本，关联到其他资料或笔记，并可从关联面板跳转到新标签页查看
- 复习算法支持 FSRS-Lite、SM-2、固定间隔
- 四档评价：忘记、困难、良好、简单
- 复习开始默认不再自动调用系统默认程序，避免干扰阅读
- 无法内嵌预览的文件可手动选择“默认打开”或“其他应用”
- 支持软件内新建/编辑笔记，也支持按本地文件方式创建 Markdown 笔记
- 笔记存储路径可自定义，支持勾选、全选、取消选择、批量导出和确认删除
- 笔记随迁移包一起备份迁移；导入迁移包后会自动修正笔记路径到当前电脑的数据目录
- 自动记录阅读时长、复习次数、历史和下次到期时间
- 支持提醒、主题、强调色、自定义 CSS、中英界面
- 设置页支持自定义导出目录、体检、数据库备份、CSV/JSON 导出、完整迁移包导出/导入；每次导出、备份和导入前备份都会弹出保存位置
- 独立“帮助”页内置文件库、复习、笔记管理、导出迁移、路径配置、插件和维护说明
- 设置页提供插件管理，可控制内置成就插件、社交插件，以及后续外部插件；插件关闭后对应界面模块会从布局中移除，启用后再动态挂载

## 安全说明

扫描文件库只读取路径和文件元数据，不移动、不复制原始资料。删除索引记录也不会删除原始文件。删除笔记会删除用户创建的 Markdown 笔记文件和数据库记录，软件会先弹出确认。

## 打包

```powershell
python -m PyInstaller --noconfirm --clean --onefile --windowed --name "智能文件复习系统2.14.0_WebUI" --add-data "web;web" --hidden-import webview.platforms.winforms app.py
```

打包完成后运行：

```powershell
dist\智能文件复习系统2.14.0_WebUI.exe
```
